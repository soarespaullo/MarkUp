# ⚡ MarkUp - HTML to Markdown Converter

O **MarkUp** é uma ferramenta web moderna e minimalista projetada para converter textos formatados e código HTML em Markdown de forma instantânea. Ideal para desenvolvedores, redatores e usuários que precisam de agilidade na criação de documentação e conteúdos para plataformas como GitHub, Notion e Obsidian.

![Licença](https://img.shields.io/badge/licen%C3%A7a-MIT-green)

## 🚀 Funcionalidades

- **Conversão Instantânea:** Basta colar o conteúdo (Ctrl+V) em qualquer lugar da tela.
- **Upload de Arquivos:** Suporte para carregar arquivos `.html` ou `.txt` diretamente.
- **Live Preview:** Visualize como o seu Markdown será renderizado em tempo real.
- **Modo Escuro/Claro:** Interface adaptável que respeita a preferência do sistema e salva sua escolha.
- **Estatísticas em Tempo Real:** Contador de palavras e caracteres integrado.
- **Exportação Facilitada:** Opções para copiar o código com um clique ou baixar como um arquivo `.md`.
- **Compartilhamento Rápido:** Envie seu conteúdo convertido diretamente para o WhatsApp.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas e leves para garantir performance máxima:

* [Bootstrap 5](https://getbootstrap.com/) - Framework CSS para um design responsivo e moderno.
* [Turndown](https://github.com/mixmark-io/turndown) - Motor de conversão de HTML para Markdown.
* [Marked](https://marked.js.org/) - Renderizador de Markdown para o modo de visualização.
* [Bootstrap Icons](https://icons.getbootstrap.com/) - Conjunto de ícones elegante.
* [Google Fonts (Inter & Fira Code)](https://fonts.google.com/) - Tipografia focada em legibilidade e leitura de código.

## 💻 Como Usar

1. Acesse o projeto em: [soarespaullo.github.io/markup](https://soarespaullo.github.io/markup) (ou seu link customizado).
2. **Cole** o conteúdo HTML desejado usando `Ctrl + V`.
3. Ou **Clique** na zona de colagem para selecionar um arquivo do seu computador.
4. Alterne entre as abas **Editor** e **Visualizar** para conferir o resultado.
5. Use os botões de ação para **Copiar**, **Baixar** ou **Compartilhar**.
